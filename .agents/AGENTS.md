# Agent Rules

- From now on, never definitively claim a bug is fixed or a feature is complete until you have explicitly verified it. If you cannot run a test to verify it yourself, you must say "I have written the code, please test it" rather than claiming "it is fixed." Furthermore, always double-check React lifecycle quirks when modifying input states.
- When I say move an element, ONLY adjust the parent container's padding/margin. Do NOT touch the internal spacing, and NEVER exaggerate values unless explicitly told to.
